# Fusedio Tap

## How do I install these formulae?

`brew install fusedio/tap/<formula>`

Or `brew tap fusedio/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "fusedio/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
